Domain Contributions

Mechanical

The robot uses a three-wheeled omni-directional drive for precise, flexible movement and body rotation. A hexagonal base evenly distributes weight, ensuring stability and smooth navigation. Vertical aluminium mounts provide a strong yet lightweight framework to secure the stick mechanism with minimal vibration. A CAD model optimized wheel, motor, and mount placement, ensuring accurate alignment and reliable performance during fabrication.

Electrical

The ESP32 Devkit V1 acts as the control hub, handling wireless communication, motor coordination, and synchronization with vision inputs. Power regulation is managed by the compact MP1584 buck converter, ensuring stable voltage supply. Motor control is achieved through the Cytron MDD20A driver, delivering high-current, efficient, and smooth operation for the DC motors.

Coding

The software integrates YOLOv8 for real-time box detection and OpenCV for precise red-circle tracking using color segmentation. Commands are sent via WebSocket to the ESP32, ensuring low-latency control. A stability-based logic locks onto the target only after consistent detection, improving precision and reducing false triggers.
